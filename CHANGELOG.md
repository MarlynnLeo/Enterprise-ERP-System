# 变更日志 (Changelog)

本文件记录 Enterprise ERP System 的所有重要变更。格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/) 规范。

---

## [1.4.0] - 2026-05-12

### 新增
- **编码规则引擎全局治理**：53 条编码规则统一由 `coding_rules` 表管理，支持前缀、日期格式、序号长度可视化配置
- **质检模板解析服务** (`InspectionTemplateResolverService`)：智能匹配物料→类目→通用模板的三级优先级
- **打印模板中心专业化**：财务报表、质检报告、采购/销售单据全面覆盖
- **16 条新增数据库迁移脚本**：权限闭环、菜单路由对齐、质检模板治理

### 修复
- 修复会计分录金额为 0 时的校验异常
- 修复 GL 唯一索引冲突 (`Duplicate entry` for `uq_active_document`)
- 修复在线时长排行榜数据解析（兼容数组/对象格式）
- 修复质量基线迁移中 `nonconforming_products` 外键引用顺序错误 (#1)
- 修复生产领料数量闭环计算

### 变更
- 11 处硬编码编号生成逻辑全部迁移至 `CodeGeneratorService` 统一引擎
- 自动生成的中文备注统一替代英文备注
- 隐藏生产计划详情中的 BOM 路径列
- 全模块 UI 一致性优化与样式统一
